# Spatial Displacement Theory (SDT)

This repository contains research papers, data, code, and figures related to Spatial Displacement Theory.

## Structure

- **Papers/** - Research papers and documentation
  - SDT_Foundation_v1.1.docx/pdf
  - Redshift_Compensator_v1.0.pdf
  - Star_Calculator_v1.0.pdf

- **Data/** - Validation and benchmark data
  - Exoplanet_Validation.csv
  - Benchmark_Results.xlsx

- **Code/** - Implementation modules
  - sdt_core/ - Core SDT functionality
  - sdt_redshift/ - Redshift compensation algorithms
  - sdt_stars/ - Star calculation utilities

- **Figures/** - Visualizations and diagrams
