## Chapter 3: Unit Conversions

**Standard SI Units:**

- Length: meter [m]
- Mass: kilogram [kg]
- Time: second [s]
- Current: ampere [A]
- Temperature: kelvin [K]

**Derived Units:**

- Force: newton [N] = kg·m/s²
- Energy: joule [J] = N·m = kg·m²/s²
- Pressure: pascal [Pa] = N/m² = kg/(m·s²)
- Frequency: hertz [Hz] = s⁻¹
